Инструкция для подключения 
1. Миграция происходит автоматически при запуске проекта(проект Main код App.cs)там надо указывать строку подключения
2. Заполнение базы данных происходит скриптом(приложен в паке)
3. в проекте Main.cs для того что бы сработали баттоны надо переписать свое мето расположение пректа
4.САМОЕ НЕУДБНОЕ! В проектах Seller и User во всех фалах .cs необходимо указывать руками строку подключения, так же указать строку подключения нужно в файле DesignTimeDbContextFactory.cs находящия в проекте Marketplace.Data и в файле appsettings.json хранящимся в App.xaml проекта Test.WPF
